# 🌍 Global COVID-19 Analytics Dashboard (Power BI)

## 📊 Project Overview
This Power BI dashboard analyzes global COVID-19 data, highlighting trends in total cases, total deaths, death rate percentage, and continent-wise distribution.

---

## 🎯 Key Insights
- Total Cases & Total Deaths Overview
- Death Rate % Analysis
- Monthly Trend Analysis
- Continent-wise Case & Death Distribution
- Country-level Death Comparison
- Global Map Visualization

---

## 🛠 Tools Used
- Power BI Desktop
- DAX
- Power Query
- Data Modeling
